Aplikasi dikembangkan berbasis mobile-first dengan menggunakan Ionic + Angular sebagai frontend
mobile, Laravel / Django Rest Framework sebagai backend API, dan MySQL sebagai basis data.
Seluruh komponen sistem harus terintegrasi penuh tanpa penggunaan dummy, dan dapat diuji
secara nyata (real-end-to-end integration).
Sistem ini wajib mendukung proses bisnis sebagai berikut:
• Event Organizer (Admin) dapat membuat event lengkap dengan form pendaftaran yang
dapat dikustomisasi, menentukan apakah event mengeluarkan sertifikat atau tidak, serta
memantau dan mengelola data peserta secara menyeluruh.
• Peserta dapat mencari event, melihat detailnya, mendaftar secara online, menerima QR
code untuk validasi kehadiran, serta mengambil sertifikat digital jika memenuhi kriteria
kehadiran.
Sistem harus mampu menghasilkan QR code secara otomatis, menyimpan dan mengelola data
peserta terverifikasi, menyediakan link verifikasi sertifikat, serta mendukung visualisasi data peserta
dalam bentuk laporan.
